## 想定されるエラーと対処

### 1. 権限エラー
```
"You do not have the appropriate permissions"
```
→ sho1ユーザーにチーム作成権限が必要

### 2. チーム存在エラー
```
"船舶専用チーム「Ocean Dream チーム」が存在せず、作成権限がありません"
```
→ 管理者にチーム作成を依頼

### 3. API接続エラー
```
"Network Error"
```
→ Mattermostサーバーが起動しているか確認
